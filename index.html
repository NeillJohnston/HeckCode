About HeckCode

I wanted to make my own minimal language that was similar to Brainfuck. HeckCode just sounds more PG.

  • < move pointer to the left
  • > move pointer to the right
  • + swap cell and storage
  • - set cell to 0 (false)
  • ! flip value of cell
  • & set cell to (cell AND storage)
  • . output cell value
  • , set cell to user input
